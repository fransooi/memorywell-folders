🎯 What is MemoryWell?

MemoryWell transforms any folder into a self-archiving workspace. Unlike complex backup systems, MemoryWell uses simple, visible folders you can browse directly in your file manager.

🚀 Quick Start

Installation

# macOS / Linux
git clone https://github.com/FrancoisLionet/memorywell-folders.git
cd memorywell-folders
./install.sh

# Windows (PowerShell)
git clone https://github.com/FrancoisLionet/memorywell-folders.git
cd memorywell-folders
.\install.ps1

Basic Usage

# Initialize a folder
cd my-project
mwinit

# Add files and archive
echo "Hello World" > file.txt
mwpush "first version"

# Continue working
echo "More content" > file2.txt
mwpush --usedelta "second version"

# Extract an archive
mwextract 00-20260314-123456-IMAGE-first-version

With GUI (Optional)

# Initialize with clickable apps
mwinit --gui

# Now double-click the apps in your folder:
# - MemoryWell-Push.app
# - MemoryWell-Extract.app
# - MemoryWell-Find.app
# - MemoryWell-SetFavorite.app

🎨 How It Works

MemoryWell creates a special folder structure:

  • 01-last-week/ - Archives from the last 7 days
  • 02-last-month/ - Archives from the last 30 days
  • 03-last-year/ - Archives from the last 365 days
  • 04-favorites/ - Your favorite archives + 00-last link
  • 05-folders/ - ALL archives (actual files)

When you push, all files at the root are moved into a timestamped archive. When you extract, the archive is restored to the root. Simple!
