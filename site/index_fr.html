🎯 Qu'est-ce que MemoryWell ?

MemoryWell transforme n'importe quel dossier en espace de travail auto-archivant. Contrairement aux systèmes de sauvegarde complexes, MemoryWell utilise des dossiers simples et visibles que vous pouvez parcourir directement dans votre gestionnaire de fichiers.

🚀 Démarrage Rapide

Installation

# macOS / Linux
git clone https://github.com/FrancoisLionet/memorywell-folders.git
cd memorywell-folders
./install.sh

# Windows (PowerShell)
git clone https://github.com/FrancoisLionet/memorywell-folders.git
cd memorywell-folders
.\install.ps1

Utilisation de Base

# Initialiser un dossier
cd mon-projet
mwinit

# Ajouter des fichiers et archiver
echo "Bonjour le monde" > fichier.txt
mwpush "première version"

# Continuer à travailler
echo "Plus de contenu" > fichier2.txt
mwpush --usedelta "deuxième version"

# Extraire une archive
mwextract 00-20260314-123456-IMAGE-premiere-version

Avec Interface Graphique (Optionnel)

# Initialiser avec applications cliquables
mwinit --hidden

# Maintenant double-cliquez sur les apps dans votre dossier :
# - MemoryWell-Push.app
# - MemoryWell-Extract.app
# - MemoryWell-Find.app
# - MemoryWell-SetFavorite.app

🎨 Comment Ça Marche

MemoryWell crée une structure de dossiers spéciale :

  • 01-last-week/ - Archives des 7 derniers jours
  • 02-last-month/ - Archives des 30 derniers jours
  • 03-last-year/ - Archives des 365 derniers jours
  • 04-favorites/ - Vos archives favorites + lien 00-last
  • 05-folders/ - TOUTES les archives (fichiers réels)

Quand vous faites push, tous les fichiers à la racine sont déplacés dans une archive horodatée. Quand vous faites extract, l'archive est restaurée à la racine. Simple !
